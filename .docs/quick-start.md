# Quick start for agents
Source of truth: `AGENTS.md`, `.plans/README.md`, `.plans/*.md`, `docs/release.md`, `docs/runbook.md`

Start here when you need to work in this repo quickly.

- Read `AGENTS.md` first for the top-level contract.
- Read `.plans/README.md` in order, then the target `.plans/*.md` file for branch name, PR title, and validation.
- Use `docs/runbook.md` for local setup, deploy recovery, secret rotation, and incident checks.
- Use `docs/release.md` for version tags and release flow.
- Keep changes scoped to one plan at a time.

If the worktree is already mixed, stop and decide whether to split the work or accept a combined PR.
